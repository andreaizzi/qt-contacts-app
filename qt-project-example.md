# Qt Project Example - Contacts

## Objective

Create a GUI application using the Qt library that implements a Contacts (telephone book).
The interface displays:
- **Last Name**: pippo
- **First Name**: pluto
- **Contact indicator**: 2/2
- **Navigation buttons**: `<` and `>`
- **Action buttons**: Load, Save, Add, Edit, Sort, Quit

---

## Interface Requirements

The interface must include:

- At minimum, two editable fields for entering and displaying the first name and last name of a contact

- A label indicating which contact is currently displayed and the total number of contacts in the address book

- Two buttons for navigating forward and backward through the contacts in the address book

- A **Load** button for loading a previously saved address book

- A **Save** button for saving the current contacts to a file

- An **Add** button for adding a new contact to the address book

- An **Edit** button for updating one of the two fields of a contact

- A **Sort** button for alphabetically sorting the contacts in the address book by last name

- A **Quit** button for closing the address book application
